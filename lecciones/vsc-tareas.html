Visual Studio Code. Tareas

VSC Tareas

  • Abrir en navegador

Abrir páginas en el navegador

  • Crear Tarea
    1. Elegir la opción de menú Tareas > Configurar tarea.
    2. Definir la tarea en el archivo task.json: 
      {
    "version": "0.1.0",
    "tasks": [
        {
            "taskName": "Abrir en Firefox",
            "command": "explorer",
            "windows": {
                "command": "C:\\Program Files\\Mozilla Firefox\\firefox.exe"
            },
            "args": [
                "${file}"
            ],
            "problemMatcher": []
        },
        {
            "taskName": "Abrir en Chrome",
            "command": "explorer",
            "windows": {
                "command": "C:\\Program Files (x86)\\Google\\Chrome\\Application\\chrome.exe"
            },
            "args": [
                "${file}"
            ],
            "problemMatcher": []
        }
    ]
}
  • Utilizar tarea:
    1. Elegir la opción de menú Tareas > Ejecutar tarea ....
    2. Elegir la tarea a aplicar.
