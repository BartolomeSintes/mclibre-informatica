Visual Studio Code. Perfiles

VSC: Perfiles

  • Importar perfiles

En febrero de 2023 Visual Studio Code introdujo una nueva funcionalidad, los perfiles (en inglés profiles). Un perfil es un conjunto de ajustes de Visual Studio Code agrupados. Estos ajustes pueden incluir extensiones, preferencias, tareas, atajos de teclado y elementos del interfaz. Podemos tener varios perfiles instalados, adaptados a diferentes necesidades y cambiar de uno a otro en cualquier momento. Podemos exportar e importar perfiles para reutilizarlos en otros ordenadores o compartirlos con otras personas. Los perfiles importados se pueden modificar posteriormente para adaptarlos a nuestros gustos o necesidades.

Los perfiles tienen algunas limitaciones a tener en cuenta:

  • En relación con las Preferencias de configuración (las que se guardan en el fichero settings.json), las preferencias que incluyen referencias a rutas y ejecutables no se pueden importar ni exportar, por lo que se deben añadir de forma manual posteriormente.
  • No se pueden añadir archivos en los perfiles. Por ejemplo, si una extensión se configura mediante un fichero que hemos creado manualmente, la extensión se puede incluir en un perfil, pero el fichero de configuración no. El fichero se debería crear de forma manual posteriormente en los ordenadores en los que importemos el perfil.
