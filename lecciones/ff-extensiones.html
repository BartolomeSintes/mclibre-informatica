En esta página se explica la instalación y uso de los complementos de Firefox.

Complementos: plug-ins, temas y extensiones

Una de las características que destacó a Firefox por encima del resto de navegadores es la gran cantidad de complementos disponibles, que permitían ampliar y personalizar Firefox. En Firefox siempre han existido diferentes tipos de complementos:

  • plug-ins, que permiten reproducir en el navegador más formatos de archivos (pdf, flash, java, etc.).
  • temas, que permiten cambiar completamente la apariencia de Firefox.
  • extensiones, que amplian las funcionalidades del navegador.

La razón por la que las extensiones de Firefox podían ser tan potentes era que las extensiones podían acceder a cualquier componente interno de Firefox. Pero en agosto de 2015, Firefox anunció que en noviembre de 2017 (como así fue, con la publicación de Firefox 57) cambiaría completamente la gestión de las extensiones, eliminando el acceso al interior de Firefox y obligando a las extensiones a acceder únicamente a través de la API WebExtensions. El objetivo era doble: por un lado, permitir que Firefox pudiera sufrir grandes cambios internos sin afectar a las extensiones y, por otro lado, facilitar el desarrollo de extensiones, puesto que la API WebExtensions es muy similar a la API de extensiones de Google Chrome.

Por ello, a partir de Firefox 57, el número y potencia de las extensiones de Firefox se ha reducido enormemente, pero también es cierto que poco a poco, la API WebExtensions va ampliándose y dando nuevas posiblidades a los creadores de extensiones, como se comenta periódicamente en el blog de complementos de Mozilla. El tiempo dirá si valió la pena el sacrificio realizado.

Los complementos se pueden conseguir:

  • desde el propio Firefox, mediante el Administrador de complementos.
  • desde el sitio web Firefox Add-ons, mantenida por la Fundación Mozilla.
  • desde las páginas web de sus creadores.

Las extensiones se distribuyen en forma de archivos con extensión xpi y su instalación es automática.

Administrador de complementos

El Administrador de complementos permite conocer los complementos disponibles, instalarlos, configurarlos o desinstalarlos.

El Administrador de complementos se puede abrir de varias formas, mediante:

  • escribiendo about:addons en la barra de dirección
  • el atajo de teclado Alt+May+A
  • el menú Herramientas > Complementos (para mostrar el menú hay que pulsar la tecla Alt):

  • el botón "Abrir menú" de la barra de dirección y eligiendo la opción Complementos:

El Administrador de complementos se abre en una nueva pestaña:

El Administrador de Complementos se divide en varias secciones a las que se accede con los iconos de la izquierda:

  • Obtener complementos: para buscar y conocer nuevos complementos e iniciar su instalación.
  • Extensiones: para ver las extensiones ya instaladas, configurarlas, desactivarlas o desinstalarlas.
  • Temas: para ver los temas ya instalados, configurarlos, desactivarlos o desinstalarlos.
  • Plug-ins: para ver los plug-ins ya instalados, configurarlos, desactivarlos o desinstalarlos.

Encontrar complementos

El blog de complementos de Mozilla ofrece cada mes una selección de extensiones recomendadas.

El mejor sitio para encontrar complementos es el sitio web Firefox Add-ons, donde podemos encontrar los complementos clasificados por muchos criterios (categorías, popularidad, valoraciones de los usuarios, novedades, colecciones temáticas, etc.) y mucha información sobre cada complemento.

Pero si ya conocemos el nombre exacto del complemento y simplemente queremos instalarlo, la forma más cómoda es utilizar el Administrador de complementos, como se explica en el apartado Instalar extensiones desde el Administrador de complementos.

Actualmente (septiembre de 2018), la sección "Obtener complementos" del Administrador de complementos muestra una información muy breve sobre unos cuantos complementos.

Instalar complementos

Las extensiones de Firefox se pueden instalar de tres formas distintas:

  • desde el Administrador de complementos
    Si se conoce el nombre exacto del complemento, esta es la opción más cómoda.
  • desde la web Firefox Add-ons (o en general, desde un sitio permitido)
    Si se conoce el nombre exacto del complemento, esta es la opción que yo recomiendo, porque la web Firefox Add-ons ofrece más información que el Administrador de complementos.
  • desde una página web cualquiera (es decir, un sitio no permitido)
    Esta es la opción menos recomendable por motivos de seguridad aunque a veces se pueden encontrar versiones más actualizadas que en Firefox Add-ons o versiones beta de los complementos.

Seguridad en la instalación: sitios permitidos

Por motivos obvios de seguridad, la instalación de extensiones en Firefox debe estar controlada.

Por ello, en el menú Herramientas > Opciones > Privacidad y Seguridad debe mantener marcada la opción "Advertirle cuando algún sitio intente instalar complementos". En ese caso, Firefox sólo permitirá la instalación de extensiones si el usuario autoriza previamente al sitio web.

Haga clic en el botón "Excepciones..." puede consultar la lista de sitios desde los que está permitido instalar excepciones.

Los sitios web oficiales incluidos de forma predeterminada son Firefox Add-ons y Firefox Test Pilot .

  • Se mostrará la lista de resultados de la búsqueda. Haga clic en la ficha de la extensión deseada:

Extensión ColorZilla

ColorZilla es un selector de colores que permite obtener el código de color de los píxeles de la pantalla. ColorZilla está incluida en Firefox Add-ons.

Actualmente (septiembre de 2018), la versión más reciente de ColorZilla para Firefox es la versión 3.3, publicada el 7 de marzo de 2017. También está disponible para Chrome la versión 2.0, publicada el 24 de diciembre de 2016.

Notas:

  • ColorZilla en Firefox tiene un problema importante y es que no permite copiar colores de una imagen si el navegador está abriendo únicamente la imagen (si la imagen está insertada en una página web, entonces sí que puede copiar un color de la imagen). ColorZilla en Chrome no tiene ese problema, aunque para ello se tiene que marcar la opción "Permitir acceso a URL de archivo" en el menú Más herramientas > Extensiones > ColorZilla
  • Firefox incluye un selector de color más simple en las herramientas de desarrollador web.

ColorZilla se instala en el extremo derecho de la barra de navegación de Firefox y de Chrome:

Para elegir el formato del código de color:

  • Haga clic en el icono de ColorZilla y en el menú que se abrirá, elija la opción Options:

  • Se abrirá una nueva pestaña mostrando las opciones de ColorZilla. Despliegue el menú "Auto-copy color format", elija el formato del código de color deseado (en este curso se recomienda elegir el formato hsl) y haga clic en el botón Save. A continuación, puede cerrar la pestaña.

Para copiar un color de la pantalla:

  • Haga clic en el icono de ColorZilla. Al mover el ratón a la página, el cursor se convierte en una cruz y se mostrará una ventana flotante:

  • Sitúe el cursor sobre el píxel cuyo color se quiere averiguar. La barra flotante de ColorZilla muestra en todo momento el color y el código de color del píxel sobre el que se encuentra el cursor (en formato RGB).

  • Haga clic en el píxel. El valor del color se mostrará durante unos segundos y después ColorZilla se cerrará. El valor del color estará copiado en el portapapeles de Windows.

  • A continuación, ese valor se puede pegar en la hoja de estilo que estemos editando.

Extensión Web Developer

Web Developer añade a Firefox una barra de herramientas con herramientas útiles para el desarrollo de páginas web. Web Developer está incluida en Firefox Add-ons.

Actualmente (septiembre de 2018), la versión más reciente de Web Developer para Firefox es la versión 2.0.1, publicada el 15 de noviembre de 2017. También está disponible para Chrome.

En cdlibre.org hay una sección dedicada a Desarrollo web > Otros, con información detallada sobre la última versión estable publicada.

Web Developer se instala en el extremo derecho de la barra de navegación de Firefox y de Chrome:

Al hacer clic en el icono de Web Developer, se despliega una ventana con todas las herramientas, organizadas en pestañas.

Al hacer clic en cada pestaña, se muestra las herramientas correspondientes. Haga clic en la herramienta que quiera aplicar.

Dependiendo de la herramienta, Web Developer puede mantener la pestaña actual o abrir una nueva pestaña. Por ejemplo, si se elige la herramienta CSS > Disable All Styles, la página se mostrará sin las propiedades de estilo. En el icono de la barra de herramientas se mostrará un número indicando el número de herramientas activas y en el menú de Web Developer se marcará la herramienta activa.

Para desactivar una herramienta, haga clic de nuevo en ella.

Validación con Web Developer

Web Developer permite la validación de varios tipos de documentos, con algunas limitaciones. Las herramientas disponibles se encuentran en l apestaña Tools del menú de Web Developer. Abra primero en Firefox la página que quiera validar y elija a continuación la herramienta correspondiente.

Las primeras opciones del menú permiten validar páginas web disponibles en Internet (http://....), mientras que las dos últimas permiten validar páginas web locales (que se abren en el navegador con la dirección file:///...).
