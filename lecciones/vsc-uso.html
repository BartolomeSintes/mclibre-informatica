Abrir carpeta

El área de trabajo de Visual Studio Code es una carpeta y todo su contenido (incluidas subcarpetas). Esa carpeta es la carpeta raíz del área de trabajo.

Para abrir una carpeta (es decir, un área de trabajo), elija la opción de menú Archivo > Abrir carpeta.

Visual Studio Code guarda la información particular de un área de trabajo en una carpeta oculta .vscode que se crea en la carpeta raíz del área de trabajado. Por ejemplo, las tareas se guardan en el archivo tasks.json de la carpeta .vscode.

Actualmente (septiembre de 2017), Visual Studio Code permite cambiar de área de trabajo simplemente abriendo la carpeta raíz correspondiente, pero sólo puede mostrar un área de trabajo. Los desarrolladores están trabajando para que el interfaz de Visual Studio Code permita mostrar y trabajar con varias áras de trabajo simultáneamente.
