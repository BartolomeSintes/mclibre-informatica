Complementos: extensiones, temas aplicaciones y juegos

Una de las características que ayuda a explicar la popularidad de Google Chrome es la gran cantidad de complementos que permiten ampliar y personalizar Chrome. Existen diferentes tipos de complementos:

  • extensiones, que amplian las funcionalidades del navegador
  • temas, que permiten cambiar la apariencia de Chrome
  • aplicaciones, que permiten instalar aplicaciones que usan el intefaz de Chrome
  • juegos, que permiten instalar juegos que usan el intefaz de Chrome

Los complementos se pueden conseguir en la Chrome web store.

Extensión ColorZilla

ColorZilla es un selector de colores que permite obtener el código RGB de los píxeles de la pantalla.

  • Colorzilla en Chrome web store
  • Página web de Colorzilla

Actualmente (octubre de 2017), la versión más reciente de Colorzilla para Chrome es la versión 2.0, publicada el 24 de diciembre de 2016.

Nota:Las siguientes capturas corresponden a Colorzilla en Firefox.

Colorzilla se instala en el extremo derecho de la barra de navegación de Firefox y de Chrome:

Para elegir el formato del código de color:

  • Haga clic en el icono de Colorzilla y elija la opción Options:

  • Despliegue el menú "Auto-copy color format", elija el formato del código de color deseado (en este curso se recomienda elegir el formato hsl) y haga clic en el botón Save. A continuación, la pestaña se puede cerrar haciendo clic en el icono de aspa.

Para copiar un color de la pantalla:

  • Haga clic en el icono de Colorzilla. Al mover el ratón a la página, el cursor se convierte en una cruz:

  • Sitúe el cursor sobre el píxel cuyo color se quiere averiguar. La barra flotante de Colorzilla muestra en todo momento el color y el código de color del píxel sobre el que se encuentra el cursor.

  • Haga clic en el píxel. El valor del color se guardará en el portapapeles de Windows.

  • A continuación ese valor se puede pegar en la hoja de estilo que estemos editando.

Extensión Web Developer

Aunque las herramientas de desarrollador que incorpora Chrome permiten analizar con detalle una página web, hay extensiones como Web Developer que incluyen herramientas de análisis muy útiles.

  • Web Developer en Chrome web store

Las capturas siguientes muestran un ejemplo de uso de la extensión Web Developer, concretamente para desactivar la hoja de estilo de una página web:

  • Abra la página web cuya hoja de estilo quiere desactivar

  • Haga clic en el icono de Web Developer (una rueda dentada). Se mostrará una ventana con todas las herramientas de Web Developer organizadas por temas en pestañas. Haga clic en la pestaña CSS. Se mostrarán todas las herramientas relacionadas con hojas de estilo. Haga clic en "Disable all styles" para desactivar los estilos y aplicar únicamente la hoja de estilo predeterminada del navegador:

  • Inmediatamente se verá la página web sin estilos:

  • El icono de Web Developer indica el número de herramientas aplicadas (en este caso, una):

  • Para volver a aplicar la hoja de estilo, simplemente actualice la página (F5).
